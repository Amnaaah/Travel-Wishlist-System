# Travel Wishlist System

## Overview
The Travel Wishlist System is a web-based application developed as the coursework of EAD2 module.  
It allows users to manage a personalized wishlist of places they wish to visit.

## Features
- User registration and account management
- Browse travel destinations reviewed by other users.
- Create, Read, Update & Delete destinations from a wishlist
- Simple and user-friendly interface
- An organized MVC architecture

## Technologies Used
- Java
- Spring Boot
- HTML
- CSS
- Thymeleaf
- MySQL 

## Project Structure
The project follows the Spring Boot MVC architecture:
- **Controller layer** – handles user requests
- **Service layer** – contains business logic
- **Repository layer** – manages database operations
- **Templates** – handles the UI views
